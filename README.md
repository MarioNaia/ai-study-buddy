
# AI Study Buddy

AI Study Buddy transforms raw study notes into useful learning material using AI.

Students can paste lecture notes and instantly generate:

• Clear summaries
• Quiz questions
• Flashcards

## Problem

Students often spend hours converting notes into study material.
This process is repetitive and slows down learning.

## Solution

AI Study Buddy automatically converts notes into structured learning formats, helping students study faster and retain information better.

## Features

- AI-powered summarization
- Quiz question generation
- Flashcard generation
- Clean responsive UI
- Accessible interface

## Tech Stack

- Next.js
- TypeScript
- TailwindCSS
- OpenAI API

## How to Run

```bash
npm install
npm run dev
```
